Accessibility & Keyboard Navigation

WCAG 2.2 AA semantics: role="grid", live aria-rowcount / aria-colcount, and the full WAI-ARIA grid keyboard pattern.

role=… aria-rowcount=… aria-colcount=…
Tab Focus enters the grid. ↑ ↓ ← → Move between cells. Home / End First / last cell in the row. Ctrl+Home / Ctrl+End First / last cell of the grid. Page Up / Page Down Scroll the viewport one page. Space Toggle selection of the focused row. Shift+Space Extend selection from the anchor. Ctrl+A Select all rows (multi mode). Enter Enter edit on the focused editable cell. Esc Cancel an in-progress edit.

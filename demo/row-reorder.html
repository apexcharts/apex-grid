Row Drag-Reorder

Enable with rowReordering = { enabled: true }, then drag any row to a new position, or use the keyboard: focus a row, press Space to grab, ↑ / ↓ to move, Space / Enter to drop, Esc to cancel. The grid holds a manual order (mutually exclusive with sorting) and emits rowMoved; the toolbar uses grid.moveRow(from, to).

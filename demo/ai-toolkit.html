AI Toolkit Enterprise

Type a request in plain language and the grid turns it into a change: sort, filter, group, and more. The <apex-grid-ai> panel sends your prompt through grid.runPrompt(), which validates the returned patch against the grid's schema, applies it with setState(), and offers a one-click Undo. Switch to Ask a question for a read-only answer. This page uses a tiny offline mock so it runs with no API key.
